Módulo 01 - Primeiros passos do HTML

(voltar)

O HTML (HyperText Markup Language) é uma linguagem de marcação que define a estrutura e o conteúdo das páginas da web. É uma linguagem de texto simples que é usada para descrever a estrutura de uma página da web, usando tags. As tags são usadas para identificar diferentes partes da página da web, como cabeçalhos, parágrafos, imagens e links. O HTML foi desenvolvido pela World Wide Web Consortium (W3C), uma organização internacional que desenvolve padrões para a web. A primeira versão do HTML foi publicada em 1991, e a versão mais recente é o HTML 5, que foi publicado em 2014. O HTML é uma linguagem muito importante para a web, pois é a linguagem usada para criar as páginas da web que vemos todos os dias. É uma linguagem fácil de aprender e usar, e é a linguagem de marcação mais usada no mundo.

Aqui estão alguns exemplos de tags HTML:

  • html - Esta tag define o início de um documento HTML.
  • head - Esta tag define a cabeçalho do documento HTML. O cabeçalho é usado para armazenar informações sobre o documento, como o título da página e as palavras-chave.
  • body - Esta tag define o corpo do documento HTML. O corpo é usado para armazenar o conteúdo da página, como texto, imagens e links.
  • h1 -Esta tag define um cabeçalho de nível 1.
  • p - Esta tag define um parágrafo.
  • img - Esta tag insere uma imagem na página. a - Esta tag cria um link.
